
# Bucketry Website - Section 1

## Project Overview
This repository contains the first section of the **Bucketry** website. This project is part of a continuous web development assignment focused on building a structured, responsive-style layout using legacy HTML techniques.

## Features Included (Section 1)
For this initial phase, the following components have been implemented:

1.  **Header & Branding:** A professional header featuring the Bucketry logo/brand name using table-based positioning.
2.  **Navigation Menu:** A fully functional horizontal navigation bar with links to Home, Collections, About, and Contact sections.
3.  **Hero Section:** A high-impact main banner area including:
    *   Main Headline (Value Proposition)
    *   Sub-headline (Description of services/products)
    *   Call-to-Action (CTA) Button for "Explore Collections"

## Implementation Details
*   **Structure:** Built entirely with standard **HTML5**.
*   **Layout:** Managed using **HTML Tables** to ensure proper alignment of the header and hero content without external CSS.
*   **Styling:** Colors, typography, and spacing are handled via HTML attributes such as `bgcolor`, `cellpadding`, and `<font>` tags.

## How to View
1. Open the `index.html` file in any standard web browser.
2. The page is designed to display a clean, structured top-of-page experience as required for Section 1.
